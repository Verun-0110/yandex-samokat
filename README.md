# Создание заказа и получение заказа по его номеру Яндек Самокат (мобильное приложение для курьера)

# Тесты на проверку создния заказа и получения заказа по его номеру Яндекс Самокат с помощью API Яндекс Самокат (мобильное приложение для курьера)

- Проект направлен на автоматизацию позитивного теста при создании заказа и получение заказа по его номеру в Яндекс Самокат, а именно что можно создать заказа, потом получить заказ по его номеру
- Окружение: PyCharm Community Edition 2023.2.2
- Для запуска тестов должны быть установлены пакеты pytest и requests
- Запуск всех тестов выполянется командой pytest
